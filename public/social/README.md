# Social icons

Place your icon files here and point to them from `src/data/social.js`.

- Default starter files: `github.svg`, `linkedin.svg`, `email.svg` (you can delete them and use `.png` instead — just update the `icon` field for each entry).
- Recommended size: about **28×28 px** to **48×48 px** (they scale with CSS).

Example after switching to PNG:

```js
icon: '/social/github.png',
```
